hexo + GitHub Pages 搭建个人博客

Posted on 2017-04-25 | In blog

本文记录在 Windows 平台下使用 hexo + GitHub Pages 搭建个人博客。并且使用个人域名。

首先，介绍一下什么是 hexo。下面是 hexo 官方的解释：

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装 Hexo

首先，确保自己电脑有安装下列应用程序：

  • Node.js
  • git

部署到 github

首先，在自己的 github 下创建一个名字为 username.github.io，注意这里的命名格式，例如我创建的仓库就是 barry-ren.github.io。

这里我们使用 SSH keys 让本地博客与 github 建立联系。

添加 SSH Keys 到 github

  1. 进入本地 C:/Users/Barry/.ssh 目录。复制 id_rsa.pub 的全部内容。

  2. 进入自己的 github 账户，选择 Settings –> SSH and GPG keys –> New SSH keys,将本地复制的 id_rsa.pub 下的内容粘贴到此即可，完后确认，这里会让你输入自己 github 账户确认一下。
