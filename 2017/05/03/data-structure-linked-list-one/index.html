循环链表

循环链表是一种链式存储结构，它的最后一个结点指向头结点，形成一个环。因此，从循环链表中的任何一个结点出发都能找到任何其他结点。循环链表的操作和单链表的操作基本一致，差别仅仅在于算法中的循环条件有所不同。

在单链表中，判别条件为 p != NULL 或 p->next != NULL,而循环单链表的判别条件为 p != L 或 p->next != L。
